## プロダクトの詳細 
自らの「投げる」というアクションによって「光」による反応が返ってくるプロダクト。  
遊び方はフリスビーと同様で、友達同士で投げて遊ぶものです。プロダクト自体が光っているので夜の公園でも遊ぶことができます。

## 使用モジュール 
- Arduino pro mini
- 3軸加速度センサ(アナログ出力)
- マイコン内臓RGBテープLED(NeoPixel)
- 写ルンです フラッシュ部分
- Arduino用 Relay Module V3
